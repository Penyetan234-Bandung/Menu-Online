    ```bash

    ```

2.  **Buka File `index.html`**
    Langsung buka file `index.html` di browser pilihan Anda.

    > Karena proyek ini menggunakan ES Modules (`import`/`export` di JavaScript), cara terbaik untuk menjalankannya secara lokal adalah dengan menggunakan **web server sederhana**. Jika Anda menggunakan Visual Studio Code, ekstensi **"Live Server"** sangat direkomendasikan.
